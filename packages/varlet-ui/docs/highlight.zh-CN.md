# IDE语法高亮

### 介绍
组件库提供了针对`webstorm`和`vscode`的组件高亮能力，通过插件和设置可以获得`语法高亮`，`代码补全`，`文档说明`。

### Webstorm
通过设置`Editor->General->Code Completion->Show the documentation popup in ___ ms`开启组件库文档说明。
